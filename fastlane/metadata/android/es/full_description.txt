<i>PCAPdroid</i> es una aplicación de código abierto que te permite monitorizar y exportar el tráfico de red de tu dispositivo. La aplicación simula una VPN para lograr una captura sin root pero, al contrario que una VPN, el tráfico se procesa localmente en el dispositivo.

<b>Características:</b>

* Registra y examina conexiones realizadas por el usuario y las aplicaciones del sistema
* Obtiene el SNI, la consulta DNS, la solicitud HTTP, la URL HTTP y la dirección IP remota
* Crea reglas para filtrar el tráfico útil y detecta fácilmente algunas anomalías
* Vuelca el tráfico en un archivo PCAP, para descargar desde un navegador, o transmitirlo a un receptor remoto para su análisis en tiempo real (por ejemplo, wireshark)
* Utiliza la aplicación en combinación con mitmproxy para descifrar el tráfico HTTPS/TLS (se necesitan conocimientos técnicos)
* En los dispositivos con root, captura el tráfico mientras se ejecutan otras aplicaciones VPN

Si piensas utilizar PCAPdroid para realizar análisis de paquetes, consulta <a href='https://emanuele-f.github.io/PCAPdroid/quick_start#14-packet-analysis'>la sección específica</a> del manual.
