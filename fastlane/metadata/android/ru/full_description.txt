<i>PCAPdroid</i> - это инструмент для мониторинга и захвата сетевого трафика с открытым исходным кодом. Он может захватывать трафик устройства Android без прав суперпользователя. Трафик можно отправить на удаленный приёмник.

<b>Особенности:</b>

* Журналирование и проверка подключений, сделанных пользователем и системными приложениями
* Извлечение SNI, DNS-запроса, URL-адреса и удалённого IP-адреса
* Применение фильтра для захвата трафика только выбранного приложения
* Лёгкая загрузка файла трафика PCAP благодаря встроенному HTTP-серверу
* Потоковая передача PCAP на удалённый приёмник для дальнейшего анализа (например, wirehark)
* Расшифровка трафика HTTPS/TLS через удаленный mitmproxy

<b>Важно</b>: PCAP, сгенерированный PCAPdroid, не является точным на 100%. Дополнительные сведения см. В разделе <a href='https://emanuele-f.github.io/PCAPdroid/quick_start#14-pcap-reliability'>Надёжность PCAP</a>.

PCAPdroid использует Android VpnService для получения всего трафика, генерируемого приложениями Android. Фактически внешний VPN не создаётся, трафик обрабатывается приложением локально.

Посетите <a href='https://github.com/emanuele-f/PCAPdroid'>хранилище Github</a> проекта для получения дополнительных сведений.
